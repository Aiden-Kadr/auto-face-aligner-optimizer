# Auto Face Aligner & Optimizer (AI Face Crop)

Автоматический скрипт на Python для обнаружения, выравнивания (поворота по оси глаз), обрезки и оптимизации фотографий лиц с последующей конвертацией в формат Base64.

Проект использует искусственный интеллект (`dlib`) для поиска анатомических точек лица и библиотеку `cv2` (OpenCV) для продвинутой графической обработки.

---

## 🚀 Основные возможности

* **Умный поворот (Выравнивание):** ИИ находит внутренние и внешние уголки глаз, рассчитывает угол наклона головы и поворачивает изображение так, чтобы линия глаз стала строго горизонтальной.
* **Автоматическая обрезка:** Скрипт аккуратно вырезает область лица из общей фотографии.
* **Стандартизация размера:** Итоговое изображение сжимается/растягивается до фиксированного формата (например, 800x600 пикселей) без потери пропорций лица.
* **Двойной экспорт:** Результат сохраняется как готовое изображение (`.jpg`), так и в виде текстовой строки (`Base64.txt`) для удобной передачи через API или сохранения в базы данных.
* **Графический интерфейс:** Удобное системное окно для выбора файла на компьютере (через Tkinter) — не нужно прописывать пути вручную в консоли.

---

## 🛠️ Технологии и требования

Проект написан на **Python 3.x** и использует следующие ключевые библиотеки:

* `opencv-python` (OpenCV) — для трансформации, поворота и изменения размера картинок.
* `dlib` — для работы нейросети (поиск лица и 68 ключевых точек).
* `tkinter` — для вызова нативного диалогового окна выбора файлов.

> **Важно для пользователей Windows:** Для успешной сборки библиотеки `dlib` в системе должны быть предварительно установлены инструменты компиляции **Visual Studio Build Tools** (с компонентом "Разработка настольных приложений на C++").

## 📦 Инструкция по установке и запуску

### 1. Клонирование репозитория
```bash
git clone [https://github.com/Aiden-Kadr/auto-face-aligner-optimizer.git](https://github.com/Aiden-Kadr/auto-face-aligner-optimizer.git)
cd auto-face-aligner-optimizer 
```
2. Установка зависимостей
Установите необходимые пакеты через менеджер пакетов pip:

``` Bash
pip install opencv-python dlib
```
3. Скачивание ИИ-модели
Для работы скрипта необходима предобученная модель распознавания лиц dlib.

Скачайте файл модели: shape_predictor_68_face_landmarks.dat (его можно найти в открытом доступе на GitHub или официальном сайте dlib).

Положите распакованный файл .dat в корневую папку проекта (туда же, где лежит ваш скрипт).
